# Speedy's Workshop

A small personal site for documenting mods, projects and progress.

## About

This repository hosts a simple GitHub Pages site for Speedy's Workshop:
- Single-page site: `index.html`
- Styling: `styles.css`

Preview (when published): https://raylit200.github.io/speedysworkshop

## Quick start

To preview locally:
1. Clone the repo:
   git clone https://github.com/raylit200/speedysworkshop.git
2. Open `index.html` in your browser:
   - double-click the file, or
   - run a simple HTTP server (recommended):
     python -m http.server 8000
     # then visit http://localhost:8000

## Make changes

Edit `index.html` and `styles.css`, then commit and push:

git add index.html styles.css
git commit -m "Update site"
git push origin main

If your repository is empty locally, run:
git branch -M main
git push -u origin main
